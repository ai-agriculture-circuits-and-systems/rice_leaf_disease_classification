# Original Data

This directory contains the original dataset files before standardization.

## Contents

- **6 disease category folders**: Original image folders with only image files (JSON files have been removed)
  - `Healthy Rice Leaf/`
  - `Bacterial Leaf Blight/`
  - `Brown Spot/`
  - `Leaf Blast/`
  - `Leaf scald/`
  - `Sheath Blight/`

- **generate_coco_annotations.py**: Original annotation generation script (kept for reference)

## Note

- All JSON annotation files have been removed from these folders
- Standardized data is located in `../rice_leaves/` directory
- Original images are preserved here for reference and backup purposes
